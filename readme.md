# Cloning in project
Go to your terminal window, and type in `git clone` and the ssh url under code at the top of the github repository (green button) e.g `git@github.com:rshanks379/Work-Experience.git`

# Troubleshooting

## I don't have git

You can either use this https://desktop.github.com/download/ or you can use the terminal. For this install using https://git-scm.com/install/windows and potentially use a client like putty or windows terminal?
